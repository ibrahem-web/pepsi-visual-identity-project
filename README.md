# مشروع الهوية البصرية لشركة بيبسي

هذا المشروع يعرض الهوية البصرية الكاملة لشركة بيبسي، ويغطي جميع عناصر العلامة التجارية بدءًا من الشعارات حتى المواد المطبوعة والرقمية.

## محتويات المشروع

### 1️⃣ الشعارات والتصميمات
- شعار الشركة الأصلي
- صور الشعار بأنواع مختلفة
- نسخة الشعار بالحروف

### 2️⃣ بطاقات العمل
- ملف تصميم بطاقة العمل بصيغة AI
- صور لبطاقة العمل النهائية

### 3️⃣ المواد التعليمية والعروض
- برشور للجامعة
- برشور خاص بمشروع الشركة

### 4️⃣ ملفات المشروع النهائية
- مشروع CD للشركة
- تصميم مغلف الشركة
- فولدر الشركة الرسمي

### 5️⃣ مواد إضافية
- فلاير الشركة
- كرتونة الشنطة الخاصة بالشركة
- ورق صور

### 6️⃣ مقال عن الشركة
- مقال تفصيلي يشرح رؤية وهوية الشركة

## حالة المشروع
✅ المشروع منتهي – يشمل كافة عناصر الهوية البصرية من تصميمات رقمية ومطبوعات.

## ملاحظات
- جميع الملفات منظمة ضمن مجلدات لتسهيل الوصول لكل عنصر من عناصر الهوية البصرية.
- يمكن استخدام المشروع كمرجع لتصميم هوية بصرية كاملة لشركات مشابهة.
